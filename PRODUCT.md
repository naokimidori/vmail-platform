# Product

## Register

product

## Users

Admin operators managing a temporary email service. They check accounts, monitor mailbox activity, generate access links, and troubleshoot delivery. Context: desk work, frequent tab-switching, speed matters.

## Product Purpose

V-Mail is the admin console for a Cloudflare-backed disposable mailbox service. Operators create accounts, inspect messages, and manage domains. The interface should feel like a precise instrument panel, not a marketing page.

## Brand Personality

Precise, quiet, efficient. The tool disappears into the task.

## Anti-references

- SaaS dashboards with gradient hero sections and marketing copy in every empty state
- Over-decorated admin panels with glassmorphism, heavy shadows, or playful illustrations
- Any interface where the chrome competes with the data

## Design Principles

1. **Earned familiarity.** Standard patterns, consistent component vocabulary, no invented affordances.
2. **Density over decoration.** Every pixel serves information or action. White space is structural, not ornamental.
3. **State-driven feedback.** Hover, focus, loading, error: each state is clear and immediate. No motion for motion's sake.

## Accessibility & Inclusion

WCAG AA. Reduced motion respected. Keyboard navigation supported throughout.
